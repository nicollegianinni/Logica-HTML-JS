TIPOS DE TAMANHOS DE LETRAS :





SERVE PARA TITULO: H1 1 POR PAGINA !



UM POUCO MENOR, O H2 PODE TER 3 POR PAGINAS.



H3 É MAIS CONVENCIONAL PARA ESCREVER TITULOS DE TEXTOS NO CORPO (BODY) OU MESMO TEXTO NO CORPO (BODY).



H4 : PARA ESCREVER EM TEXTOS EM GERAL.



H5 SERVE PARA ESCREVER TEXTOS MENORES.


BEM PEQUENO, O H6 É USADO PARA ESCREVER AQUELES LETRAS MIUDAS NO CONTRATO. (RSRS)
